# Remove-QuickAssist

## SYNOPSIS
.
Function wrapper for removing Windows QuickAssist

## SYNTAX

```
Remove-QuickAssist [<CommonParameters>]
```

## DESCRIPTION
Removes the Windows Quick Assist app using the Remove-WindowsCapability cmdlet

## EXAMPLES

### EXAMPLE 1
```
Remove-QuickAssist
```
